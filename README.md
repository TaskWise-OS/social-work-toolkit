# Australian Social Work Practice Toolkit

A modern, interactive web application providing evidence-informed tools and frameworks for professional social workers in Australia.

## What It Does

This application helps social workers identify the right tools, frameworks, and resources for their practice by:

- Providing personalized recommendations based on practice needs
- Ensuring alignment with Australian legislation and state/territory requirements
- Offering trauma-informed, evidence-based guidance
- Supporting decision-making across multiple practice areas

## Features

- **20+ Evidence-Based Tools**: Comprehensive library of assessment frameworks, screening tools, intervention approaches, and practice frameworks
- **Intelligent Matching**: Smart relevance scoring based on practice area, jurisdiction, and query keywords
- **Trauma-Informed Focus**: Highlights trauma-informed tools and approaches
- **Australian Context**: All tools aligned with Australian legislation and best-practice standards
- **Modern UI**: Stunning blue and white neon dark mode design with smooth animations

## Practice Areas Covered

- Child Protection
- Mental Health
- Family Services
- Disability Support
- Aged Care
- Youth Services
- Domestic Violence
- Substance Use

## How to Use

1. Open `index.html` in a web browser
2. Describe your practice need in the text area
3. Optionally select practice areas and state/territory
4. Click "Find Recommended Tools"
5. Review personalized recommendations with practice guidance

## Technology

- Pure HTML, CSS, and JavaScript
- No backend required
- No API keys needed
- Runs entirely in the browser

## Design

- Neon blue and white dark mode color scheme
- Glassmorphism effects
- Smooth animations and transitions
- Fully responsive design

## For Professional Use

This toolkit is designed for qualified social workers practicing in Australia. Always consult relevant legislation, organizational policies, and professional supervision when making practice decisions.
